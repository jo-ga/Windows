Requisitos del sistema operativo Windows 2000

Windows 2000 era un sistema operativo para empresas y para ejecutar servidores de red o los servidores de archivo

-Procesador Pentium o compatible a 133 MHz o superior
-RAM 64 MB de memoria RAM, más memoria generalmente mejora el rendimiento
-Disco Duro 2 GB de espacio en el disco duro con 650 MB libres
-CD-ROM Unidad de CD-ROM o DVD-ROM

Caracteristicas de windows 2000

Windows 2000 fue sucedido por nuevos sistemas operativos de Microsoft. La línea Windows 2000 Servidor ha sido reemplazada por Windows Server 2003, y Windows 2000 Professional con Windows XP Professional. La familia de sistemas operativos Windows 2000 hizo su avance desde la Fase de Soporte Principal a la Fase de Soporte Extendido el 30 de junio de 2005. Microsoft dice que esto marca la progresión del producto a través de su Directiva de Ciclo de Vida de Productos.
